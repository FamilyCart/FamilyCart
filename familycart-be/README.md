```
python 3.13.9
virtualenv
```

```
Python 3.13.9

```
### Installing

A step by step series of examples that tell you how to get a development env running.

```
clone the project:
git clone https://github.com/FamilyCart/familycart-be.git OR git@github.com:FamilyCart/familycart-be.git

get in to project
cd familycart-be

create virtual environment
virtualenv . --python=python3.13.2 #note: you can follow any type of virtual environment creation like conda, pyenv etc

Activate Virtual Environment

source bin/acctivate

Install required packages from requirements/v1/requirements.txt
pip install -r requirements/v1/requirements.txt

```
